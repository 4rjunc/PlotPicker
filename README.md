## Run Locally

Set up virtual environment

```zsh
python3 -m venv venv
```

Activate virtual environment

```zsh
source venv/bin/activate
```

Install dependencies

```zsh
pip3 install -r requirements.txt
```

Run Streamlit app

```zsh
streamlit run app.py
```
